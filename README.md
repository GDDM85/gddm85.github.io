# danielmcdermott
Personal site of Daniel McDermott — Lead Developer building scalable, real-world systems.

This repository now contains an Astro-powered profile and career hub website.

## Development

- `npm install`
- `npm run dev`

## Build

- `npm run build`
- `npm run preview`

## Deployment

The GitHub Actions workflow in `.github/workflows/pages.yml` builds the site and publishes the `dist/` output to GitHub Pages.
